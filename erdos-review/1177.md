# Review: Erdos Problem 1177

**Status:** OPEN

## Original Problem

Let $G$ be a finite 3-uniform hypergraph, and let $F_G(\kappa)$ denote the collection of 3-uniform hypergraphs with chromatic number $\kappa$ that do not contain $G$ as a subhypergraph.

Three conjectures (Erdos, Galvin, Hajnal):

1. If $F_G(\aleph_1) \neq \emptyset$, then $\exists X \in F_G(\aleph_1)$ with $|X| \leq 2^{2^{\aleph_0}}$
2. If $F_G(\aleph_1) \neq \emptyset$ and $F_H(\aleph_1) \neq \emptyset$, then $F_G(\aleph_1) \cap F_H(\aleph_1) \neq \emptyset$
3. If $\kappa, \lambda$ are uncountable and $F_G(\kappa) \neq \emptyset$, then $F_G(\lambda) \neq \emptyset$

## Current Formalization

```lean
theorem chromatic_three_uniform_hypergraph_conjectures :
    answer(sorry) ↔
      ∀ (V : Type*) [Fintype V] (G : Set (Finset V)),
        (∀ e : Finset V, e ∈ G → e.card = 3) →
        ∃ (W : Type*) (H : Set (Finset W)),
          (∀ e : Finset W, e ∈ H → e.card = 3) := by
  sorry
```

| Aspect | Captured? | Notes |

### Critical issues

1. **Trivially true (Critical):** The statement says: given any finite 3-uniform hypergraph G, there exists some 3-uniform hypergraph H. This is trivially true — take H = ∅ (the empty hypergraph on any vertex set W). The entire statement has no mathematical content.

2. **None of the three conjectures formalized (Critical):** The original poses three specific conjectures involving chromatic numbers, cardinality bounds, intersections, and cardinal transfer. None of these appear in the formalization:
   - **Conjecture 1** (cardinality bound): Not present
   - **Conjecture 2** (intersection non-emptiness): Not present
   - **Conjecture 3** (cardinal transfer): Not present

3. **No chromatic number:** The definition of $F_G(\kappa)$ requires chromatic number $= \kappa$. Chromatic number of hypergraphs does not appear in the formalization.

4. **No subhypergraph avoidance:** $F_G(\kappa)$ consists of hypergraphs *not containing G*. The formalization places no containment constraint between G and H.

5. **No cardinality constraints:** Conjecture 1 requires $|X| \leq 2^{2^{\aleph_0}}$. Conjecture 3 involves uncountable cardinals $\kappa, \lambda$. None of these appear.

### What would be needed

A faithful formalization would require:
- Chromatic number for hypergraphs (minimum colors for a proper vertex coloring)
- Subhypergraph containment (injection preserving hyperedges)
- The family $F_G(\kappa)$ defined in terms of both
- Then each conjecture stated separately or as a conjunction

## Verdict

**Confidence: 3%**

The formalization is trivially true and captures none of the three conjectures' content. The 3-uniform hypergraph representation is correct in isolation, but the statement merely asserts that given a 3-uniform hypergraph, another 3-uniform hypergraph exists — which is vacuously true. All chromatic number conditions, subhypergraph avoidance, cardinality bounds, and intersection properties are absent.
