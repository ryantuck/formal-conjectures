# Review of Erdos Problem 573

## Problem Statement
Is it true that $\mathrm{ex}(n; \{C_3, C_4\}) \sim (n/2)^{3/2}$?

## Lean Formalization
The formalization uses `extremalNumber` with a predicate for avoiding graphs.

```lean
theorem extremal_triangle_four_cycle (answer : Prop) :
    answer ↔ (fun (n : ℕ) => (extremalNumber n (fun G => avoidsC3C4 G) : ℝ)) ~[Filter.atTop]
      (fun n => ((n : ℝ) / 2) ^ (3 / 2)) := by
  sorry
```

## Critique
The formalization is correct.
- `avoidsC3C4` correctly checks for lack of triangle and 4-cycle.
- The asymptotic equivalence $\sim$ is captured by `~[Filter.atTop]`.
- The value $(n/2)^{3/2}$ is correctly transcribed.
