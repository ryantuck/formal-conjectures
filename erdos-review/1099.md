# Erdős Problem 1099 Review

## Problem Statement
The problem asks if $\liminf_{n\to \infty}h_\alpha(n) \ll_\alpha 1$, where $1=d_1<\cdots<d_{\tau(n)}=n$ are the divisors of $n$, and for $\alpha>1$,
\[h_\alpha(n) = \sum_{i=1}^{\tau(n)-1} \left( \frac{d_{i+1}}{d_i}-1\right)^\alpha.\]

Source: [erdosproblems.com/1099](https://www.erdosproblems.com/1099)

## Formalization Review
The formalization in `FormalConjectures/ErdosProblems/1099.lean` defines the function `h` as:
```lean
noncomputable def h (α : ℝ) (n : ℕ) : ℝ :=
  let ds := n.divisors.sort (· ≤ ·)
  (List.zip ds ds.tail).map (fun (d1, d2) => ((d2 : ℝ) / d1 - 1) ^ α) |>.sum
```
This correctly implements the sum over adjacent divisors. The list `ds` represents the sorted divisors $d_1, \dots, d_{\tau(n)}$. `List.zip ds ds.tail` creates pairs $(d_i, d_{i+1})$, and the map function computes the term $(\frac{d_{i+1}}{d_i}-1)^\alpha$.

The conjecture is stated as:
```lean
theorem divisor_gap_bound (α : ℝ) (hα : 1 < α) :
    ∃ (C : ℝ),  atTop.liminf (fun n => h α n) ≤ C
```
This accurately captures the statement $\liminf_{n\to \infty}h_\alpha(n) \ll_\alpha 1$. The existence of $C$ inside the scope of $\alpha$ allows the bound to depend on $\alpha$. The use of `atTop.liminf` is appropriate for the limit inferior as $n \to \infty$.
