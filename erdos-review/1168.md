# Review: Erdos Problem 1168

**File:** `FormalConjectures/ErdosProblems/1168.lean`
**Source:** [erdosproblems.com/1168](https://www.erdosproblems.com/1168)
**Status:** OPEN

## Original Problem

Prove that $\aleph_{\omega+1} \nrightarrow (\aleph_{\omega+1}, 3, \ldots, 3)_{\aleph_0}^2$ without assuming GCH.

This means: there exists a 2-coloring of pairs from $\aleph_{\omega+1}$ using $\aleph_0$ colors such that:
- Color 0 has no monochromatic set of size $\aleph_{\omega+1}$
- Each other color has no monochromatic set of size 3

The key requirement is proving this **independently of GCH**.

## Current Formalization

```lean
def PartitionRelationFails (α : Type*) (r : ℕ) (gamma : Type*) (targets : gamma → Type*) : Prop :=
  ∃ (coloring : Finset α → gamma),
    ∀ (color : gamma) (H : Set α),
      (∃ (f : targets color → α), Function.Injective f ∧ Set.range f ⊆ H) →
      ∃ (s : Finset α), s.card = r ∧ (↑s : Set α) ⊆ H ∧ coloring s ≠ color

theorem partition_relation_fails_aleph_omega_plus_one :
    ∀ (α : Type*) [Infinite α],
    ∀ (gamma : Type*) [Countable gamma],
      PartitionRelationFails α 2 gamma (fun _ => Fin 3 ⊕ α) := by
  sorry
```

## Analysis

### What the formalization captures

| Aspect | Captured? | Notes |
|--------|-----------|-------|
| Negative partition relation | Yes | `PartitionRelationFails` correctly negates `PartitionRelation` |
| 2-element subsets | Yes | `r = 2` |
| Countably many colors | Partially | `[Countable gamma]` (includes finite) |
| Statement format | Yes | Direct statement to prove |
| Category/AMS tag | Yes | `@[category research open, AMS 03]` |

### Critical issues

1. **Uniform targets distort the problem (Critical):** The formalization uses `fun _ => Fin 3 ⊕ α` for ALL colors. This gives each color a target of size $3 + |\alpha|$. For infinite $\alpha$, this equals $|\alpha|$. So the formalization says: no color has a monochromatic set of size $|\alpha|$.

   The original says: color 0 needs no monochromatic set of size $\aleph_{\omega+1}$, while colors $1, 2, 3, \ldots$ need no monochromatic set of size 3. The formalization is **strictly weaker** than the original for the non-special colors — it allows monochromatic 3-element sets in all colors, as long as there's no monochromatic $|\alpha|$-set.

2. **Missing cardinality constraints (Critical):** $\alpha$ should have cardinality $\aleph_{\omega+1}$ specifically, not just be infinite. $\gamma$ should be countably infinite ($\aleph_0$), not just countable.

3. **"Without GCH" not encoded:** The original demands the result hold in ZFC without GCH. The formalization makes no reference to set-theoretic axioms.

4. **Universal quantification too broad:** `∀ (α : Type*) [Infinite α]` claims the result for ALL infinite types. The original is specifically about $\aleph_{\omega+1}$.

### Good aspects

- The `PartitionRelationFails` definition is a correct logical negation of the partition relation.
- The docstring honestly documents the missing cardinality constraints.

## Verdict

**Confidence: 15%**

The negation structure is correct, but the uniform target `Fin 3 ⊕ α` fails to capture the heterogeneous nature of $(\aleph_{\omega+1}, 3, 3, \ldots)$, and the cardinality constraints are absent. The statement as written is about a different (weaker) partition relation than the original.
