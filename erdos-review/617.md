# Review of Erdos Problem 617

## Problem Statement
If edges of $K_{r^2+1}$ are r-colored, exist $r+1$ vertices with at least one color missing on induced $K_{r+1}$.

## Lean Formalization
The formalization asserts this property.

```lean
theorem erdos_617 (r : ℕ) ...
    ∃ (S : Finset V) (k : Fin r),
      S.card = r + 1 ∧
      ∀ u ∈ S, ∀ v ∈ S, u ≠ v → coloring s(u, v) ≠ k := by
  sorry
```

## Critique
The formalization is correct.
- $r+1$ vertices: `S.card = r + 1`.
- Color $k$ missing: `coloring ... ≠ k`.
