# Review of Erdos Problem 610

## Problem Statement
Is $\tau(G) \le n - \omega(n)\sqrt{n}$ for clique transversal number?

## Lean Formalization
The formalization asks for existence of such $\omega$.

```lean
noncomputable def cliqueTransversal (G : SimpleGraph α) : ℕ := sorry

theorem clique_transversal_bound (answer : Prop) :
    answer ↔ ∃ (ω : ℕ → ℝ), ... (cliqueTransversal G : ℝ) ≤ (n : ℝ) - ω n * Real.sqrt n
```
