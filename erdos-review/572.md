# Review of Erdos Problem 572

## Problem Statement
Show that for $k \ge 3$,
$$ \mathrm{ex}(n; C_{2k}) \gg n^{1+1/k}. $$

## Lean Formalization
The formalization asks for a lower bound.

```lean
theorem extremal_even_cycle_lower_bound (k : ℕ) (hk : k ≥ 3) :
    ∃ c : ℝ, c > 0 ∧ ∀ᶠ (n : ℕ) in Filter.atTop,
      (extremalNumber n (cycleGraph (2 * k)) : ℝ) ≥ c * (n : ℝ) ^ (1 + 1 / (k : ℝ)) := by
  sorry
```

## Critique
The formalization correctly captures the lower bound condition ($\gg$ usually means $\ge c \cdot$ for some constant $c > 0$, or sometimes order of magnitude, but here lower bound is the specific claim).
- `cycleGraph (2 * k)` is $C_{2k}$.
- The exponent $1 + 1/k$ is correct.
