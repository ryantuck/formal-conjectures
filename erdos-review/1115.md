# Review: Erdos Problem 1115

**Problem:** Path growth in entire functions of finite order
**Source:** [erdosproblems.com/1115](https://www.erdosproblems.com/1115)
**Status:** DISPROVED (by Gol'dberg and Eremenko, 1979)

## Source Problem

Let f(z) be an entire function of finite order, and let Gamma be a rectifiable path on which f(z) -> infinity. Let l(r) denote the length of Gamma within the disc |z| < r, and M(r) = max_{|z|=r} |f(z)|.

Questions:
1. Find a path where l(r) grows as slowly as possible
2. Estimate l(r) in terms of M(r)
3. Can such a path Gamma exist where l(r) = o(r)?

Gol'dberg and Eremenko proved that for any phi(r) -> infinity, there exists an entire function f such that log M(r) << phi(r)(log r)^2, and no rectifiable path Gamma on which f -> infinity satisfies l(r) = o(r). This disproved the possibility of always finding such slow-growing paths.

### 1. Overall structure is correct (POSITIVE)

- Universally quantifies over phi(r) -> infinity
- Existentially produces an entire function f
- States the growth bound log M(r) <= phi(r)(log r)^2
- Negates the existence of a suitable path

### 2. M(r) definition uses iSup over a non-compact type (MODERATE)

Since entire functions are continuous and the circle is compact, the supremum is attained, but the `iSup` formulation using a dependent pair (z : C) (_ : ||z|| = r) may not behave well without establishing that the set is nonempty and bounded. This is a minor technical concern for the proof rather than a statement-level issue.

### 3. Missing finite order condition on f (MODERATE)

The source problem specifies that f should be an entire function of **finite order**. The formalization requires `Differentiable â„‚ f` (entireness) and the growth bound `log M(r) <= phi(r)(log r)^2`, which for slowly growing phi implies that f has order at most 2 (since log M(r) is controlled by (log r)^2 up to a slowly growing factor). However, the Gol'dberg-Eremenko result actually constructs functions of **any prescribed finite order**, so the growth bound alone does not explicitly encode the finite order condition. The growth bound is part of the conclusion, not a hypothesis, so the finite order condition is implicitly captured but only for order roughly <= 2. This is acceptable as a formalization of the specific Gol'dberg-Eremenko result.
