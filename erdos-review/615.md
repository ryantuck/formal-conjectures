# Review of Erdos Problem 615

## Problem Statement
Does $\exists c > 0$ s.t. $\ge (1/8-c)n^2$ edges $\implies K_4$ or independent set $\Omega(n/\log n)$? (Disproved)

## Lean Formalization
The formalization negates the statement.

```lean
theorem not_ramsey_turan_bound :
    ¬ ∃ c > 0, ...
      G.edgeSet.ncard ≥ Nat.floor ((1 / 8 - c) * n^2) →
      (∃ (f : Fin 4 ↪ Fin n), ...) ∨
      (∃ (S : Finset (Fin n)), ... S.card ≥ Nat.floor ((n : ℝ) / Real.log n))
```

## Critique
The formalization is correct.
- Captures the edge density condition.
- Captures the disjunction ($K_4$ or large independent set).
