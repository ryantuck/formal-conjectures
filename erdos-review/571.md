# Review of Erdos Problem 571

## Problem Statement
Show that for any rational $\alpha \in [1,2)$ there exists a bipartite graph $G$ such that
$$ \mathrm{ex}(n;G) \asymp n^\alpha. $$

## Lean Formalization
The formalization uses `extremalNumber` (stub) and asks for $G$ such that bounds hold asymptotically.

```lean
noncomputable def extremalNumber (n : ℕ) (G : SimpleGraph α) : ℕ := sorry

theorem turan_exponent_realization (α : ℚ) (hα : 1 ≤ α ∧ α < 2) :
    ∃ (G : SimpleGraph ℕ), IsBipartite G ∧
      ∃ c₁ c₂ : ℝ, c₁ > 0 ∧ c₂ > 0 ∧
        ∀ᶠ (n : ℕ) in Filter.atTop,
          c₁ * (n : ℝ) ^ (α : ℝ) ≤ (extremalNumber n G : ℝ) ∧
          (extremalNumber n G : ℝ) ≤ c₂ * (n : ℝ) ^ (α : ℝ) := by
  sorry
```

## Critique
The formalization accurately captures the problem.
- $\mathrm{ex}(n;G)$ corresponds to `extremalNumber n G`.
- The condition $\alpha \in [1,2)$ is correct.
- The asymptotic relation $\asymp n^\alpha$ (Theta notation) is correctly expanded into lower and upper bounds with constants $c_1, c_2$.
